2 - Explain when a database would be used. Explain when a text file would be used.

A - A database should be used whenever you have data that you will need to quickly search or add to / remove from and use beyond the current session. A text file should be used whenever you need to load data into a program but you don't need to search it, or manipulate that data beyond the current session.

3 - Describe one difference between SQL and other programming languages.

A - SQL is declarative, meaning that you write what you want to do, you don't tell the system how to do it.

4 - In your own words, explain how the pieces of a database system fit together a high level.

A - The columns are categories, one of those will be the key. each row if a different item or instance in the database. Where the row meets the column is a cell, which holds a value of that column's specific data type which can be entered for future perusal.

5 - Explain the meaning of TABLE, ROW, COLUMN and VALUE.

	Table - the overall datastructure containing rows, columns and values.
	Row - Usually visualized horizontally, contains each instance or item in the table
	Column - Usually visualized vertically, defines the data type (name of values) contained in the sections of rows directly below.
	Value - Data entered into cells
